# PySpark Data Analysis and Machine Learning Projects

## Overview

This repository contains **9 weeks of PySpark practice exercises and projects**, covering:

- Data preprocessing and analysis
- Filtering and selection operations
- Aggregations and group operations
- Window functions, ranking, and quartiles
- Machine learning models: Decision Trees, Linear Regression, KMeans, Random Forests
- PySpark UDFs for custom transformations
- Dataset integration and joining (Netflix, world population, California housing, wine, Iris dataset)
